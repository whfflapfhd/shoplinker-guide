기본 테이블 - col

Mac OS의 Safari브라우저는 caption태그 다음에 colgruop의 col요소에 width값을 %로 지정하면 너비값이 제대로 처리되지 않습니다.

이를 해결하기 위해 caption안의 텍스트를 <i class="ir">텍스트</i>로 감싸줍니다.

숨겨진 행 펼쳐보기

테이블 내에서 추가정보가 담긴 행을 숨기거나 보이게 하는 기능입니다.

테이블 내 toggle버튼이 위치할 td에 <a href="#" class="ui-btn-toggle"><i class="ir">view</i></a>를 추가합니다.

숨김 컨텐츠는 toggle버튼이 있는 행 다음에 위치 시키고, tr요소에 toggle-content class를 추가합니다.

테이블
# 항목 2 항목 3 항목 4
view 데이터 2 데이터 3 데이터 4
숨겨진 추가정보
view 데이터 2 데이터 3 데이터 4
숨겨진 추가정보 

                            <table>
                                <caption><i class="ir">숨겨진 행 펼쳐보기</i></caption>
                                <thead>
                                    <tr>
                                        <th scope="col">#</th>
                                        <th scope="col">항목 2</th>
                                        <th scope="col">항목 3</th>
                                        <th scope="col">항목 4</th>
                                    </tr>
                                </thead>
                                <tbody>
                                    <tr>
                                        <td><a href="#" class="ui-btn-toggle"><i class="ir">view</i></a></td>
                                        <td>데이터 2</td>
                                        <td>데이터 3</td>
                                        <td>데이터 4</td>
                                    </tr>
                                    <tr class="toggle-content">
                                        <td colspan="4">숨겨진 추가정보</td>
                                    </tr>
                                </tbody>
                            </table>

스크롤 테이블

  • 스크롤 테이블의 thead는 접근성을 위해 필요하지만 생략해도 UI에 문제가 되지는 않습니다.

  • 스크롤 테이블의 항목셀에 sort기능의 UI를 추가하시려면 항목의 텍스트를 <span>태그로 감싸주세요
  • 스크롤 테이블의 자세한 내용은 Javascript 페이지의 Table탭을 참고해주세요.
