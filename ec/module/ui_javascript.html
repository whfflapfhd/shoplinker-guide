호출 함수 EcUi.msgPop(title,dialogType,options,message);
전달인자(argument) Title
Index
0
설명
dialog에서 보여질 title 값으로 String Type
Dialog Type
Index
1
설명

dialog의 컨텐츠를 구성할 방식을 정합니다.

단순 텍스트를 보여줄 alert 혹은 confirm 형식은 값으로 'msg'를 전달해주면 됩니다.

입력폼 등의 서식등을 보여주기 위해서는 Ajax로 호출할 페이지의 URL을 String형식으로 입력합니다.
Options
Index
2
설명

Dialog의 버튼등 옵션을 전달합니다. Type은 Object입니다.

반영할 option을 변수에 지정하고 해당 변수를 값으로 전달합니다.

전달할 옵션이 없으면 null 입렵 후 전달
Message
Index
3
설명

Dialog에 보여줄 텍스트를 입력합니다. Type은 String

입력서식 등 Ajax 호출방식에서는 생략 합니다.

Dialog - Alert

알림 메세지 형식의 Dialog 호출 방식입니다.

EcUi.msgPop("Alert Message","msg",null,"테스트 메세지");

Dialog - Confirm

Confirm 형식의 Dialog 호출 방식입니다.

EcUi.msgPop("Alert Message","msg",confirmOpt,"적용하시겠습니까?");

                                        <a href="#" class="ui-btn" onclick='EcUi.msgPop("Alert Message","msg",confirmOpt,"적용하시겠습니까?");return false'>View Dialog</a>
                                        <script>
                                            var confirmOpt = {
                                                buttons : {
                                                    "confirm" : function(){
                                                        alert("OK");
                                                    },
                                                    "close" : function(){
                                                        $(this).dialog("close");
                                                    }
                                                }
                                            }
                                        </script>

Scroll Table

스크롤 테이블 Options

scrollY

테이블 Max Height 설정 (Default : null)


                                                                <script>
                                                                    (".ui-scroll-table").fixedTable({scrollY : Number})
                                                                </script>
minWidth

테이블 Min Width 설정 (Default : null)


                                                                <script>
                                                                    (".ui-scroll-table").fixedTable({minWidth : Number})
                                                                </script>
fixedCell

가로 스크롤 시 고정할 column 설정 (Default : null)


                                                                <script>
                                                                    (".ui-scroll-table").fixedTable({fixedCell : Number})
                                                                </script>
Option 적용 예시

최소 가로 사이즈 1200px, 최대 세로 사이즈 500px, 1번, 2번 column 고정이 필요한 테이블 만들기


                                                                <script>
                                                                    (".ui-scroll-table").fixedTable({minWidth : 1200, scrollY : 500, fixedCell : 2});
                                                                    
                                                                    /* or */

                                                                    var opt = {minWidth : 1200, scrollY : 500, fixedCell : 2};
                                                                    (".ui-scroll-table").fixedTable(opt);
                                                                </script>
